# Luk vlasac

lat. Allium schoenoprasum

Luk vlasac je višegodišnja začinska biljka iz porodice lukova (Alliaceae), s finim, šupljim listovima blagog okusa luka. Cijenjena je zbog jestivih listova i mirisnih cvjetova.

Porijeklo

Jugoistočna i središnja Europa, Sibir.

Alternativni nazivi: Šnitlah, drobnjak, vlasac
